# 🎬 Movie Recommendation System

A Movie Recommendation System built using Machine Learning, Collaborative Filtering, Flask, HTML, and CSS. This application recommends movies similar to a user's favorite movie based on user ratings data from the MovieLens dataset.

---

## 🚀 Features

* Recommend movies based on user preferences
* Collaborative Filtering using Cosine Similarity
* Flask-based web application
* Clean and responsive user interface
* Real-time movie recommendations
* Easy to run locally

## ⚙️ How It Works

1. Load movie and ratings data.
2. Merge datasets using Movie IDs.
3. Create a User-Movie Matrix.
4. Calculate Cosine Similarity between movies.
5. Accept a movie title from the user.
6. Return the top 5 most similar movies.

This project uses the MovieLens Latest Small Dataset provided by GroupLens Research.

Dataset contains:

* 100,836 Ratings
* 9,742 Movies
* 610 Users
